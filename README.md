# 🍽️ EuroACE - Showcooking

Sistema de gestão de fichas técnicas para showcooking do projeto EuroACE.

## 📋 Sobre o Projeto

O EuroACE Showcooking é uma plataforma que permite ao administrador criar e gerenciar fichas técnicas de receitas, que são exibidas em um frontend público. O sistema conta com autenticação JWT, upload de imagens e um banco de dados SQLite.

### Funcionalidades

- ✅ **Autenticação de Admin** - Login seguro com JWT
- ✅ **CRUD de Fichas Técnicas** - Criar, listar e deletar receitas
- ✅ **Upload de Imagens** - Envio de fotos dos pratos
- ✅ **Frontend Admin** - Painel com formulário completo
- ✅ **Frontend Público** - Exibição das receitas para usuários
- ✅ **API REST** - Endpoints documentados

## 🛠️ Tecnologias Utilizadas

### Backend
- **Node.js** - Runtime JavaScript
- **Express** - Framework web
- **SQLite3** - Banco de dados leve
- **JWT** - Autenticação
- **Bcrypt** - Hash de senhas
- **Multer** - Upload de arquivos
- **CORS** - Compartilhamento entre frontends
